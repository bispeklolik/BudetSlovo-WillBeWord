# «Слово» — продуктовый роадмап

_Цель (2026-06-20): красивый продуктовый вид + фичи уровня конкурентов. Ниже: что делается автономно сейчас, и что отложено (требует участия пользователя — ключи/решения/железо)._

**Релиз v0.1.5 (2026-06-21)** — выпущен (тег v0.1.5, CI собирает установщик). Вошло всё перечисленное ниже, включая обезличивание и облачный Claude.

## ✅ Сделано автономно (заход 2026-06-20)

**Визуальный вид (главный «прорыв»):**
- Шрифт **Inter Variable** (полная кириллица), загрузка из пакета — офлайн.
- Палитра teal + зелёный, выверенные отступы, радиусы, тени, hover-состояния карточек.
- **SVG-иконки** вместо эмодзи; полировка кнопок, карточек, модалок, плеера, волны.
- Светлая/тёмная — обе выверены.
- **Приветственный экран** первого запуска (значок, обещание приватности, импорт, drag-drop).
- **Фирменный знак-волна** в шапке и на приветствии — айдентика.

**Фичи конкурентов (без внешних зависимостей):**
- Экспорт субтитров **SRT/VTT** (для видео-авторов — есть у Descript/Happy Scribe).
- **Поиск** по тексту записи (Ctrl+F + видимая кнопка).
- Экспорт **саммари/тезисов/лучших мыслей в docx**.
- Панель **«Лучшие мысли»** — список выделенного, клик → переход к месту + воспроизведение.
- **«Скопировать текст»** с именами говорящих в буфер обмена.
- **«Найти и заменить»** по всему транскрипту (массово чинит повторяющиеся ошибки
  распознавания, напр. имя клиента); откатывается одним Ctrl+Z, +5 тестов.
- **Глобальный поиск по всем записям** на главной (число совпадений + превью,
  клик → запись с активным поиском и переходом к найденному); +4 теста.
- **Счётчик слов** в шапке редактора (с русским склонением).
- **Иконка приложения и установщика** — знак-волна, генерируется скриптом без
  тяжёлых зависимостей (встроенный zlib → PNG-в-ICO); electron-builder вшивает её
  в exe, инсталлятор и деинсталлятор.
- Экран **настроек** (модель, ИИ-статус, тема, шпаргалка по клавишам).
- Папки: переименование папок, **перетаскивание записей мышью** в папки/крошки.
- Меню **«✨ ИИ»** — все ИИ-действия в одном месте, прогресс прямо на кнопке.
- **ИИ-обезличивание** — тумблер «Обезличено», подсветка замен, редактируемый список,
  обезличенный экспорт/копирование; всегда локально (гейт приватности); +тесты.
- **Облачный ИИ Claude по API-ключу** — выбор движка и модели в настройках, ключ хранится
  локально; чистка/саммари/мысли через выбранный движок; обезличивание остаётся локальным.
- **Удаление записи** (в Корзину, восстановимо) + «открыть папку данных» в настройках.

## ⏳ Осталось автономного (мелочи)
- Полная **статистика записи** (число говорящих, реплик, средний темп речи) — низкий приоритет.
- Больше **горячих клавиш** (скорость цифрами) — низкий приоритет.
- Понятные ошибки при отсутствии движка расшифровки (часть пути к 1.0, см. ниже).

## 🎯 Путь к «1.0 для других» (самодостаточность — главный блок)

Сейчас приложение работает только на этой машине: жёсткие пути `D:\STT` (движок),
`D:\Apps\ollama` (локальный ИИ), `D:\Apps\slovo-data` (данные). Чтобы запускалось у других:

1. **Убрать жёсткие пути D:** — данные в стандартную папку пользователя (с миграцией существующих,
   чтобы НЕ потерять текущие записи), движок/Ollama — поиск + настройка пути. *~M.*
2. **Встраивание/докачка движка и моделей** при первом запуске (Faster-Whisper + модель;
   опционально локальный ИИ). *~L, главный труд.*
3. **Мастер первого запуска** — проверить/поставить зависимости, выбрать папку данных. *~M.*
4. **Закалка надёжности** — понятные состояния при сбоях, битые файлы, сквозной end-to-end,
   проверка автообновления. *~M.*
5. **Тест на чужой/чистой машине** — без этого «работает у других» не утверждать.

**Нужны твои решения перед стартом:** (а) способ доставки модели — встроить в установщик
(большой .exe) или докачивать при первом запуске; (б) можно ли переносить твои данные из
`D:\Apps\slovo-data` в стандартную папку (с миграцией); (в) доступ ко второй машине для теста.

## Принципы качества
- Локально и приватно по умолчанию (запись не покидает ПК); облако — опция по ключу.
- Не выдумывать (ИИ): подсветка неуверенного/подозрительного, всё откатываемо.
- Каждая фича с тестами где применимо; перед «готово» — проверка.
