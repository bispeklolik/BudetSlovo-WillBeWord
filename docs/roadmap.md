# «Слово» — продуктовый роадмап

_Цель (2026-06-20): красивый продуктовый вид + фичи уровня конкурентов. Ниже: что делается автономно сейчас, и что отложено (требует участия пользователя — ключи/решения/железо)._

## Делается автономно (этот заход)

**Визуальный вид (главный «прорыв»):**
- Шрифты Figtree (заголовки) + Noto Sans (тело) — оба с кириллицей; загрузка из пакета.
- Палитра teal + зелёный (стиль «Accessible & Ethical»), выверенные отступы, радиусы, тени.
- SVG-иконки вместо эмодзи; полировка кнопок, карточек, модалок, плеера, волны.
- Светлая/тёмная — обе выверены; состояния (пусто/загрузка/ошибка) красивые.

**Фичи конкурентов (без внешних зависимостей):**
- Экспорт субтитров **SRT/VTT** (для видео-авторов — есть у Descript/Happy Scribe).
- **Поиск** по тексту записи (и по всем записям).
- **Найти и заменить** в транскрипте.
- Экспорт **саммари/тезисов/лучших мыслей в docx**.
- Панель **«Лучшие мысли»** — список выделенного, клик → переход к месту.
- Экран **настроек** (модель, ИИ вкл/выкл, тема, провайдер).
- Папки: переименование папок, перетаскивание записей мышью.
- Больше **горячих клавиш** + подсказка-шпаргалка.
- Онбординг/первый запуск.

## Отложено — нужен ты (ключи / решения / железо)

| Фича | Что нужно от тебя | Оценка |
|---|---|---|
| **Облачная расшифровка** (Groq Whisper-v3 / OpenAI) | API-ключ; согласие на облако | M–L |
| **Облачный ИИ** (Claude API) для чистки/саммари | API-ключ Claude | M |
| **Встраивание движков** (Faster-Whisper + llama.cpp + докачка моделей) — MVP-блокер для распространения | решение по размеру сборки/способу; тест на чужой машине | L (~2–3 дня) |
| **Тюнинг диаризации** (разметка говорящих) | прогон/оценка на твоих записях | M |
| **Голосовая правка через микрофон** (#16) | дизайн UX + микрофон | L |
| **Экспорт в Obsidian / Telegram-лонгрид** | формат/шаблон под тебя | M |
| **Лицензирование/оплата, мобильное приложение, совместный доступ** | продуктовые решения | L+ |

## Принципы качества
- Локально и приватно по умолчанию (запись не покидает ПК); облако — опция по ключу.
- Не выдумывать (ИИ): подсветка неуверенного/подозрительного, всё откатываемо.
- Каждая фича с тестами где применимо; перед «готово» — проверка.
