# «Слово» — продуктовый роадмап

_Цель (2026-06-20): красивый продуктовый вид + фичи уровня конкурентов. Ниже: что делается автономно сейчас, и что отложено (требует участия пользователя — ключи/решения/железо)._

## ✅ Сделано автономно (заход 2026-06-20)

**Визуальный вид (главный «прорыв»):**
- Шрифт **Inter Variable** (полная кириллица), загрузка из пакета — офлайн.
- Палитра teal + зелёный, выверенные отступы, радиусы, тени, hover-состояния карточек.
- **SVG-иконки** вместо эмодзи; полировка кнопок, карточек, модалок, плеера, волны.
- Светлая/тёмная — обе выверены.
- **Приветственный экран** первого запуска (значок, обещание приватности, импорт, drag-drop).
- **Фирменный знак-волна** в шапке и на приветствии — айдентика.

**Фичи конкурентов (без внешних зависимостей):**
- Экспорт субтитров **SRT/VTT** (для видео-авторов — есть у Descript/Happy Scribe).
- **Поиск** по тексту записи (Ctrl+F + видимая кнопка).
- Экспорт **саммари/тезисов/лучших мыслей в docx**.
- Панель **«Лучшие мысли»** — список выделенного, клик → переход к месту + воспроизведение.
- **«Скопировать текст»** с именами говорящих в буфер обмена.
- **«Найти и заменить»** по всему транскрипту (массово чинит повторяющиеся ошибки
  распознавания, напр. имя клиента); откатывается одним Ctrl+Z, +5 тестов.
- **Счётчик слов** в шапке редактора (с русским склонением).
- Экран **настроек** (модель, ИИ-статус, тема, шпаргалка по клавишам).
- Папки: переименование папок, **перетаскивание записей мышью** в папки/крошки.
- Меню **«✨ ИИ»** — все ИИ-действия в одном месте, прогресс прямо на кнопке.

## ⏳ Осталось автономного (следующий заход)
- Больше **горячих клавиш** (скорость цифрами, Esc-закрытие панелей) + расширить шпаргалку.
- Полная **статистика записи** (число говорящих, реплик, средний темп речи).
- **Иконка приложения/установщика** (.ico из знака-волны) — для панели задач и инсталлятора.

## Принципы качества
- Локально и приватно по умолчанию (запись не покидает ПК); облако — опция по ключу.
- Не выдумывать (ИИ): подсветка неуверенного/подозрительного, всё откатываемо.
- Каждая фича с тестами где применимо; перед «готово» — проверка.
