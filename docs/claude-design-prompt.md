# Промт для Claude Design — интерфейс «Слово»

---

Ты — старший продуктовый дизайнер. Спроектируй интерфейс десктопного приложения **«Слово»**. Выдай макеты экранов в **светлой и тёмной** темах, на русском языке (используй точные строки из раздела «Тексты UI»), с состояниями компонентов. Стек — на твой выбор (React + Tailwind или статичные HTML/CSS-макеты); приложение десктопное (Electron, окно ~1100–1400px шириной), не мобильное.

## Что это за продукт
«Слово» — **приватный редактор расшифровок аудио и видео**: запись разговора превращается в точный текст, **не покидая компьютер пользователя** (всё считается локально). Это «локальный Descript для русского»: импорт записи → расшифровка с прогрессом → редактор, где **аудио синхронно с текстом** (клик по слову перематывает запись, при воспроизведении слова подсвечиваются «караоке») → правка на месте → экспорт в Word/Markdown/текст.

## Для кого и какое ощущение
Основная аудитория — **психологи и консультанты**, работающие с конфиденциальными записями сессий; шире — все, кто работает голосом (юристы, преподаватели, журналисты). Материал чувствительный, поэтому интерфейс должен ощущаться **спокойным, профессиональным, заслуживающим доверия и сфокусированным** — НЕ игривым, НЕ «AI-хайповым». Приватность — не мелкий текст, а считываемое ощущение надёжности.

## Дизайн-система (соблюдать)
- **Стиль:** доступный и этичный. Высокий контраст, крупный читаемый текст (тело ≥16px), чёткие фокус-кольца (3–4px), полная клавиатурная навигация, WCAG AA+ (для длинных сессий чтения это критично).
- **Палитра (светлая):** поверхности — почти белые, чуть тёплые (#FFFFFF / #F7F8F8); основной интерактивный цвет — спокойный teal **#0891B2** (ховеры/ссылки/активные элементы); акцент успеха/готово — зелёный **#059669**; основной текст — глубокий сине-серый **#164E63 / #0F172A**; приглушённый текст — **#475569**; границы — мягкие #E2E8F0.
- **Палитра (тёмная):** поверхности — глубокий графит **#0F172A / #1E293B**; teal-акцент светлее **#22D3EE**; зелёный **#34D399**; текст #E2E8F0 / приглушённый #94A3B8; границы — белый с низкой непрозрачностью, но **видимые**.
- **Семантика уверенности (ЗАРЕЗЕРВИРОВАНЫ — нигде больше не использовать как украшение):** слова, в которых модель не уверена, подсвечиваются фоном:
  - **жёлтый/янтарный** (#FEF3C7 фон / #92400E текст в светлой; полупрозрачный янтарь в тёмной) — «недоуверенно» (порог ~уверенность < 0.72),
  - **красный** (#FEE2E2 фон / #991B1B текст) — «низкая уверенность» (< 0.5).
  - Правленные пользователем слова теряют подсветку (человек проверил). Покажи **легенду** этих цветов в редакторе.
- **Типографика:** заголовки — **Figtree**, тело — **Noto Sans** (оба с полной поддержкой кириллицы). Тело 16px, line-height 1.6–1.75; длина строки в тексте расшифровки — комфортная для чтения (≈70–90 знаков).
- **Иконки:** только SVG (Lucide/Heroicons), без эмодзи.
- **Избегать:** фиолетово-розовых «AI»-градиентов, неона, тяжёлой анимации, переусложнённого glassmorphism. Переходы 150–300ms, уважать prefers-reduced-motion.

## Экраны (спроектировать)

**1. Главная / список записей + импорт.**
Шапка с названием «Слово» и переключателем темы. Крупная основная кнопка «Импортировать запись» (принимает аудио И видео). Сетка карточек записей: название, длительность, дата. Пустое состояние: «Пока нет ни одной записи» + подсказка выбрать аудио- или видеофайл. Спокойно, много воздуха.

**2. Импорт и расшифровка (прогресс).**
После выбора файла — панель с фазами: «Проверяю и готовлю файл» → «Извлекаю звук из видео» (если видео) → «Чистка звука» → «Диаризация» → «Распознавание — NN%» → «Сохранение». Прогресс-бар (определённый там, где есть %, индикатор там, где нет), таймер, кнопка «Отменить». Настройки до старта: «Говорящих на записи» (Двое: психолог и клиент / Один / Определить автоматически), чекбокс «Чистка звука (помогает с тихой и неразборчивой речью)». Внизу — спокойная плашка-обещание: «Всё происходит на вашем компьютере: запись никуда не отправляется.»

**3. Синхронный редактор — ГЛАВНЫЙ экран (сделай его героем макета).**
- **Шапка:** название записи + статус («файл починен при импорте · сохранено»), справа кнопки «Перераспознать» и «Экспорт ▾».
- **Лента беседы (одна колонка):** реплики сгруппированы по говорящим. Слева у реплики — **метка роли** (Психолог / Клиент), цветная и **переименуемая**, рядом таймкод. Текст реплики — крупный, читаемый; внутри — подсветка неуверенных слов (жёлтый/красный, см. семантику). При воспроизведении текущее слово подсвечивается «караоке» (teal-акцентом), лента мягко автоскроллит. Клик по слову = перемотка к этому моменту. Правка слова — инлайн; у правленого слова тултип «исходно: …».
- **Нижняя панель плеера (закреплена внизу):** кликабельная **звуковая волна** с плейхедом и цветными полосами говорящих; кнопки −5с / Играть·Пауза / +5с; текущее время / общее; выбор скорости (0.75–2×). Space = play/pause.
- Покажи легенду цветов уверенности и состояние «слово выделено/редактируется».

**4. Экспорт (выпадающее меню или диалог).**
Варианты: «Word (.docx)» с переключателем «с подсветкой неуверенных слов», «Markdown (.md)», «Текст (.txt)». Коротко и понятно.

**5. (на будущее) Панель ИИ — спокойный боковой блок (показать как «скоро»/задизейблено).**
Кнопки: «Краткое содержание», «Тезисы», «Убрать слова-паразиты», «Шаблон заметки (психолог)». Подчеркнуть: для конфиденциальных записей обработка локальная.

## Ключевые требования UX
- Длинные записи (2–3 часа = десятки тысяч слов): лента должна выглядеть так, будто легко прокручивается (виртуализация) — без визуального «перегруза».
- Клавиатура: Space (play/pause), стрелки/шорткаты перемотки, undo/redo. Видимые фокус-стейты.
- Состояния: загрузка, пусто, ошибка («В файле нет звуковой дорожки»), идёт расшифровка, нет интернета — и это нормально (всё локально).
- Цвет никогда не единственный индикатор (для уверенности дай и иконку/подчёркивание).

## Тексты UI (использовать дословно, без «lorem»)
Импортировать запись · Перераспознать · Экспорт · Расшифровать запись · Расшифровка идёт · Чистка звука · Распознавание · Сохранение · Отменить · Говорящих на записи · Двое: психолог и клиент · Один (монолог) · Определить автоматически · Чистка звука (помогает с тихой и неразборчивой речью) · Всё происходит на вашем компьютере: запись никуда не отправляется · Психолог · Клиент · Светлая тема / Тёмная тема · Краткое содержание · Тезисы · Убрать слова-паразиты.

## Что отдать
Светлая и тёмная темы; **экран редактора — крупно и проработанно** (это сердце продукта); главная и панель прогресса; набор состояний компонентов (реплика, слово-уверенность, плеер, кнопки). Подпиши палитру и шрифты.
