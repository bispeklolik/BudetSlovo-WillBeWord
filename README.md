# Слово

Локальный редактор расшифровок аудио для Windows: импорт записи → распознавание речи с разделением на говорящих → синхронный редактор (аудио ↔ текст, караоке-подсветка, пометка неуверенных мест) → правка → экспорт в Word / Markdown / текст. Всё работает офлайн, записи не покидают компьютер.

> Личный инструмент. Распознавание выполняется локальным движком **Faster-Whisper-XXL** (`D:\STT`), который ставится отдельно; приложение — графическая оболочка над ним.

## Разработка

```bash
npm install
npm run dev      # запуск в режиме разработки
npm run dist     # сборка установщика (Windows, NSIS) в dist/
```

Данные приложения (проекты, аудио, расшифровки) хранятся в `D:\Apps\slovo-data` и в репозиторий не попадают.

## Релизы и автообновление

Установщик публикуется в GitHub Releases автоматически: пуш тега `vX.Y.Z` запускает сборку (GitHub Actions, `.github/workflows/release.yml`). Установленное приложение проверяет обновления при запуске (`electron-updater`).

```bash
npm version patch        # поднять версию + создать тег
git push --follow-tags   # запустить публикацию релиза
```
