# Слово — запись разговора становится текстом и смыслом

Для всех, чья работа состоит из разговоров: психологов, юристов, преподавателей, журналистов, продажников. «Слово» превращает запись встречи или сессии в аккуратную расшифровку с разделением голосов, а затем — в нужный вам результат: саммари, протокол, заметку сессии, супервизорский разбор или текст по вашему собственному промту.

**Попробовать прямо в браузере (без установки):** откройте веб-версию, перетащите запись — и получите текст. Ключи сервисов хранятся только в вашем браузере, страница работает без сервера.

**Приложение для Windows** — полная версия: локальное распознавание (запись не покидает компьютер), синхронный редактор «аудио ↔ текст», голосовая правка, обезличивание имён, конспекты, экспорт в Word/Markdown/субтитры. [Скачать последнюю версию](https://github.com/bispeklolik/BudetSlovo-WillBeWord/releases/latest).

## Что умеет

- **Расшифровка** — локально (Faster-Whisper, диаризация, приватно) или облаком на выбор: ElevenLabs Scribe (максимальная точность), Deepgram (цена/качество), OpenAI и Groq Whisper.
- **«Сделать из текста»** — библиотека готовых промтов под применения: заметка сессии, супервизорский разбор, протокол встречи, юридическое резюме, конспект лекции, цитаты для статьи, follow-up после звонка и другие. Плюс свои промты с сохранением в карточки.
- **Текстовый ИИ на выбор** — локальная модель (Ollama), Claude по ключу или OpenRouter (один ключ — сотни моделей).
- **Синхронный редактор** — клик по слову перематывает аудио, караоке-подсветка, правка слов, разрез и склейка реплик, скорость с клавиш, поиск и замена, отмена.
- **Голосовая правка** — плохо распознанную реплику можно просто надиктовать заново с микрофона.
- **Обезличивание** — локальный ИИ находит имена, места и организации и заменяет их нейтральными подстановками; оригинал не изменяется.
- **Статистика записи** — баланс речи участников (кто сколько говорил), чистая речь, слова и реплики.
- **Экспорт** — Word (docx) с подсветкой неуверенных мест, Markdown, текст, субтитры SRT/VTT, аудио.

## Приватность

Приложение локально по умолчанию: распознавание и обезличивание работают на вашем компьютере, записи никуда не отправляются. Облачные движки — осознанный выбор: включаются вручную, работают по вашему личному ключу, и приложение честно показывает, куда уходит запись.

## Разработка

```bash
npm install
npm run dev       # десктоп в режиме разработки
npm run dev:web   # веб-версия (vite)
npm run test      # vitest
npm run dist      # установщик Windows (NSIS) в dist/
```

Данные приложения хранятся отдельно от кода и в репозиторий не попадают. Установщик публикуется в GitHub Releases автоматически по пушу тега `vX.Y.Z`; установленное приложение обновляется само (electron-updater). Веб-версия собирается из `web/` и деплоится на GitHub Pages по пушу в `main`.
